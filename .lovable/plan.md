# Fix: Uncouple Terminal button does nothing\n\n## Root cause analysis\n\nI traced the failure end-to-end. Three independent bugs interact to make the button feel dead:\n\n### Bug 1 — Worker crashes on boot with `supabaseKey is required`\n`src/workers/telemetry.worker.ts` reads `import.meta.env.VITE_SUPABASE_ANON_KEY`, but the project's `.env` only defines `VITE_SUPABASE_PUBLISHABLE_KEY`. `createClient(url, undefined)` throws synchronously at worker module load → the runtime error overlay/red banner appears, which on some renders intercepts pointer events on top of the sidebar (so clicks on the Uncouple button never reach React).\n\n### Bug 2 — IndexedDB `NotFoundError: object store was not found
